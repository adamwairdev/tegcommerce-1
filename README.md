## requirement
* git
* npm (> = 5.x)
* nodejs (> = 8.x)
* reactjs
* graphql
* mongodb

## installation
1. `make sure you have installed` [nodejs](https://nodejs.org/en/download/) 
1. `git clone git@github.com:tegcommerce/tegcommerce`
1. `cd tegcommerce`
1. `sudo npm install`
1. `sudo npm start`
1. `open` [http://localhost:3000/](http://localhost:3000)
